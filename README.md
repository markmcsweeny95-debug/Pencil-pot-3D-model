# Pencil-pot-3D-model
A simple Pen pot model i have been working on to develop my CAd skills, feel free to 3d print the model if you want. Parts designed to be standardised and screw together to create adjustable sizes for different sizes of pencils or to improve stability. Holds pencils internally as well as externally to make good use of limited desk space.
